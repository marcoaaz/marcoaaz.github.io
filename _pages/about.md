Hopefully you will find useful some of the software that I wrote and regularly use for analysing images during my latest research projects. They are free and open-source in order to be sustainable. Further updates that increase the number of options, reliability and performance will be driven by user feedback and my available resources. 

Everyone is welcome to contribute to build this expanding research field.


Latest news
------
We ran a workshop (see Portfolio tab) to teach how to use the standalone software tools and reach a wider audience. The latest update of the seminar material has been uploaded to a data sharing [**service**](https://sggmp25_workshop.fromsmash.com/seminar-material-updated) (expiring 20-Mar-2026).

The latest versions of the software (25-Feb-2026) for Windows 11 can be downloaded from:

 - [**Phase Interpreter v1.1**](https://zenodo.org/records/18707639)
 - [**Chemistry Simplifier v1.1**](https://zenodo.org/records/18706790)
 - [**Cube Converter v1.1**](https://zenodo.org/records/18706798)

You are welcome to try them for your own projects and help us develop the software.
